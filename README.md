# Real-time positions of SF-Muni Map

#Features Developed
- See the realtime SF-Muni in every 15s.
- Transitions for vehicle updates
- Select SF-Muni Route from the dropdown on right.
- Hover over the bus or the stop to see its detail.

### Run the app: 

```sh
npm install
npm start
```

or

```sh
yarn
yarn start
```
